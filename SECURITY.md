# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by emailing the maintainers or opening a private issue. We will respond as quickly as possible and keep your report confidential until a fix is released.

## Supported Versions
- Only the latest version is supported for security updates.

## Security Best Practices
- Do not share sensitive information in public issues or pull requests.
- Keep dependencies up to date.
- Review code for potential security issues before merging.

Thank you for helping keep this project secure!
